# Spring_e-commerce_backend
A personal backend project created by me to improve my skills. It allows user to register, add products to cart and create an order.

## Features:
  - User registration with data validation
  - Email sending to confirm registration
  - Authentication with Json Web Token
  - Authorization roles (different privileges for admin and customer)
  - Displaying products and adding them to cart
  - Create order based on products in cart
  - HATEOAS to define control logic available to client
  - Seeding databases for developing and testing purposes

## Main technologies used in this project:
  - Spring framework
    - HATEOAS
    - Json Web Token (authentication)
    - Mail
    - JUnit
  
  - Data and dabases:
    - Hibernate
    - MySql
